## CDN Installation



For quick prototyping or non-bundler environments, use the UMD build via CDN. Use these URLs for the latest published version:

### unpkg
```html
<link rel="stylesheet" href="https://unpkg.com/vistaview/main/dist/style.css" />
<script src="https://unpkg.com/vistaview/main/dist/vistaview.umd.js"></script>
```

### jsDelivr
```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/vistaview/main/dist/style.css" />
<script src="https://cdn.jsdelivr.net/npm/vistaview/main/dist/vistaview.umd.js"></script>
```

## Bundle Sizes

VistaView is lightweight and optimized for production. Bundle sizes are for v2:

- **Core Library:** ~42KB ESM (~10KB gzip)
- **CSS:** ~7KB (~1.7KB gzip)
- **Total:** ~49KB (~11.7KB gzip)

## Browser Support

VistaView works in all modern browsers:

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Framework Integrations

VistaView provides official bindings for popular frameworks:

- **React** – Component and hook
- **Vue 3** – Component and composable
- **Svelte** – Component and hook
- **Solid** – Component and hook
- **Vanilla JS** – Direct API

See the [Integration guides](/integrations/overview) for framework-specific instructions. Each integration is maintained in the monorepo and may have its own documentation and stability guarantees.
