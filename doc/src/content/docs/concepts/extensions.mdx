Extensions add powerful features to VistaView without bloating the core library.

## Available Extensions

### UI Extensions

#### Download Button

Adds a download button to save the current high-resolution image.

```js
import { vistaView } from 'vistaview';
import { download } from 'vistaview/extensions/download';
import 'vistaview/style.css';

vistaView({
  elements: '#gallery a',
  extensions: [download()],
  controls: {
    topRight: ['download', 'zoomIn', 'zoomOut', 'close'],
  },
});
```

**Size**: 1.65 KB ESM / 1.58 KB UMD

#### Image Story

Display rich HTML content alongside images with XSS protection.

```js
import { vistaView } from 'vistaview';
import { imageStory } from 'vistaview/extensions/image-story';
import 'vistaview/style.css';
import 'vistaview/styles/extensions/image-story.css';

vistaView({
  elements: [
    {
      src: '/image1.jpg',
      alt: 'Beautiful landscape',
      story: '<h2>Mountain Vista</h2><p>Captured at sunrise...</p>',
    },
  ],
  extensions: [imageStory()],
});
```

**Size**: 33.60 KB ESM / 25.28 KB UMD

### Video Platform Extensions

All video extensions automatically:
- Show loading indicator while thumbnail loads
- Disable zoom and download controls for videos
- Embed responsive iframes

#### YouTube

```js
import { youtubeVideo } from 'vistaview/extensions/youtube-video';

vistaView({
  elements: '#gallery a',
  extensions: [youtubeVideo()],
});
```

```html
<a href="https://www.youtube.com/watch?v=dQw4w9WgXcQ">
  <img src="/video-thumb.jpg" alt="Video title" />
</a>
```

**Size**: 3.35 KB ESM / 14.28 KB UMD

#### Vimeo

```js
import { vimeoVideo } from 'vistaview/extensions/vimeo-video';

vistaView({
  elements: '#gallery a',
  extensions: [vimeoVideo()],
});
```

```html
<a href="https://vimeo.com/123456789">
  <img src="/video-thumb.jpg" alt="Video title" />
</a>
```

**Size**: 2.91 KB ESM / 13.90 KB UMD

#### Other Video Platforms

- **Dailymotion**: `dailymotionVideo()` - 3.05 KB ESM / 14.03 KB UMD
- **Wistia**: `wistiaVideo()` - 3.18 KB ESM / 14.13 KB UMD
- **Vidyard**: `vidyardVideo()` - 2.99 KB ESM / 13.97 KB UMD
- **Streamable**: `streamableVideo()` - 2.95 KB ESM / 13.94 KB UMD

### Map Extensions

All map extensions:
- Show loading indicator while map loads
- Disable zoom and download controls for maps
- Support interactive maps with pan/zoom

#### Google Maps

```js
import { googleMaps } from 'vistaview/extensions/google-maps';

vistaView({
  elements: '#gallery a',
  extensions: [
    googleMaps({
      apiKey: 'YOUR_GOOGLE_MAPS_API_KEY',
    }),
  ],
});
```

```html
<a href="https://maps.google.com/?q=40.7128,-74.0060" data-vistaview-alt="New York City">
  <img src="/map-thumb.jpg" alt="Map" />
</a>
```

**Size**: 3.91 KB ESM / 14.78 KB UMD

#### Mapbox

```js
import { mapbox } from 'vistaview/extensions/mapbox';

vistaView({
  elements: '#gallery a',
  extensions: [
    mapbox({
      accessToken: 'YOUR_MAPBOX_ACCESS_TOKEN',
    }),
  ],
});
```

```html
<a href="mapbox://40.7128,-74.0060,12" data-vistaview-alt="New York City">
  <img src="/map-thumb.jpg" alt="Map" />
</a>
```

**Size**: 5.48 KB ESM / 16.07 KB UMD

#### OpenStreetMap

Free alternative with no API key required!

```js
import { openstreetmap } from 'vistaview/extensions/openstreetmap';

vistaView({
  elements: '#gallery a',
  extensions: [openstreetmap()],
});
```

```html
<a href="osm://40.7128,-74.0060,12" data-vistaview-alt="New York City">
  <img src="/map-thumb.jpg" alt="Map" />
</a>
```

**Size**: 5.28 KB ESM / 15.87 KB UMD

### Developer Extensions

#### Logger

Debug extension that logs all lifecycle events to the console.

```js
import { logger } from 'vistaview/extensions/logger';

vistaView({
  elements: '#gallery a',
  extensions: [logger()],
});
```

**Size**: 0.61 KB ESM / 0.76 KB UMD

## Using Multiple Extensions

Combine extensions to add multiple features:

```js
import { vistaView } from 'vistaview';
import { download } from 'vistaview/extensions/download';
import { youtubeVideo } from 'vistaview/extensions/youtube-video';
import { vimeoVideo } from 'vistaview/extensions/vimeo-video';
import { googleMaps } from 'vistaview/extensions/google-maps';
import 'vistaview/style.css';

vistaView({
  elements: '#gallery a',
  extensions: [
    download(),
    youtubeVideo(),
    vimeoVideo(),
    googleMaps({ apiKey: 'YOUR_API_KEY' }),
  ],
  controls: {
    topLeft: ['indexDisplay'],
    topRight: ['download', 'zoomIn', 'zoomOut', 'close'],
    bottomLeft: ['description'],
  },
});
```

## UMD Usage

Extensions are also available as UMD bundles for CDN usage:

```html
<script src="https://unpkg.com/vistaview/dist/vistaview.umd.js"></script>
<script src="https://unpkg.com/vistaview/dist/extensions/download.umd.js"></script>
<script src="https://unpkg.com/vistaview/dist/extensions/youtube-video.umd.js"></script>

<script>
  VistaView.vistaView({
    elements: '#gallery a',
    extensions: [
      VistaViewExtensions.download(),
      VistaViewExtensions.youtubeVideo(),
    ],
    controls: {
      topRight: ['download', 'zoomIn', 'zoomOut', 'close'],
    },
  });
</script>
```

## Extension Options

Some extensions accept configuration options:

```js
// Google Maps with custom options
googleMaps({
  apiKey: 'YOUR_API_KEY',
  mapType: 'satellite', // 'roadmap', 'satellite', 'hybrid', 'terrain'
  zoom: 15,
})

// Mapbox with custom options
mapbox({
  accessToken: 'YOUR_TOKEN',
  style: 'mapbox://styles/mapbox/streets-v11',
  zoom: 12,
})

// OpenStreetMap with custom options
openstreetmap({
  zoom: 13,
  maxZoom: 18,
})
```
