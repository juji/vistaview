VistaView is highly customizable through CSS variables and comes with 17 built-in themes.

## CSS Import

VistaView CSS is separate from the JavaScript bundle:

```js
import 'vistaview/style.css';
```

## Built-in Themes

VistaView includes 17 pre-designed themes. Import a theme after the base styles:

```js
import 'vistaview/style.css';
import 'vistaview/styles/dark-rounded.css';
```

### Available Themes

#### Dark Themes

- **`dark-rounded`** - Dark theme with rounded corners and animated navigation
- **`midnight-ocean`** - Deep navy blues with cyan accents
- **`midnight-gold`** - Elegant black background with gold accents
- **`ember-glow`** - Dark theme with orange glow effects
- **`neon-nights`** - Cyberpunk-inspired magenta and cyan with glowing effects
- **`stark-minimal`** - Pure black and white, zero border radius, minimalist

#### Light Themes

- **`paper-light`** - Light theme with subtle shadows and paper-like appearance
- **`soft-neutral`** - Warm beige and cream tones with maximum roundness
- **`ice-crystal`** - Cool light blue palette with crisp styling

#### Colorful Themes

- **`autumn-amber`** - Warm oranges and browns with earthy tones
- **`cotton-candy`** - Playful pastel pink with soft styling
- **`forest-moss`** - Earthy greens inspired by nature
- **`green-lake`** - Calming teal and green palette with smooth transitions
- **`lavender-fields`** - Soft purple palette with gentle animations
- **`mint-chocolate`** - Brown and mint green combination
- **`retro-arcade`** - Bold primary colors with classic arcade aesthetics
- **`strawberry`** - Vibrant pink and red gradient theme

### Theme Comparison

```js
// Modern dark with rounded corners
import 'vistaview/styles/dark-rounded.css';

// Minimalist black & white
import 'vistaview/styles/stark-minimal.css';

// Vibrant and playful
import 'vistaview/styles/neon-nights.css';

// Elegant and sophisticated
import 'vistaview/styles/midnight-gold.css';
```

## CSS Custom Properties

Customize VistaView by overriding CSS variables:

### Colors

```css
:root {
  --vvw-bg-color: #000000;        /* Background color */
  --vvw-text-color: #ffffff;      /* Text color */
  --vvw-accent-color: #007bff;    /* Accent/highlight color */
  --vvw-border-color: #333333;    /* Border color */
}
```

### Background Effect

```css
:root {
  --vvw-bg-blur: 10px;           /* Background blur amount */
  --vvw-bg-opacity: 0.8;         /* Background opacity */
}
```

### Animation

```css
:root {
  --vvw-anim-dur: 333;           /* Animation duration (ms, unitless) */
}
```

### Layout

```css
:root {
  --vvw-init-z: 1000;            /* Initial z-index */
  --vvw-border-radius: 8px;      /* Border radius for controls */
  --vvw-control-size: 44px;      /* Size of control buttons */
}
```

## Custom Theme Example

Create your own theme by defining CSS variables:

```css
/* my-custom-theme.css */
:root {
  /* Colors */
  --vvw-bg-color: #1a1a2e;
  --vvw-text-color: #eee;
  --vvw-accent-color: #0f3460;
  --vvw-border-color: #16213e;
  
  /* Background */
  --vvw-bg-blur: 15px;
  --vvw-bg-opacity: 0.95;
  
  /* Animation */
  --vvw-anim-dur: 400;
  
  /* Layout */
  --vvw-border-radius: 12px;
  --vvw-control-size: 48px;
}

/* Custom button styling */
.vvw-control button {
  background: linear-gradient(135deg, var(--vvw-accent-color), #e94560);
  box-shadow: 0 4px 12px rgba(233, 69, 96, 0.3);
}

.vvw-control button:hover {
  transform: scale(1.1);
  box-shadow: 0 6px 16px rgba(233, 69, 96, 0.5);
}
```

Import your custom theme:

```js
import 'vistaview/style.css';
import './my-custom-theme.css';
```

## Per-Instance Styling

Style specific gallery instances using inline styles or CSS classes:

```js
const gallery1 = vistaView({
  elements: '#gallery1 a',
  initialZIndex: 1000,
});

const gallery2 = vistaView({
  elements: '#gallery2 a',
  initialZIndex: 2000,
});
```

Or add custom classes:

```html
<style>
  .my-gallery {
    --vvw-accent-color: #ff6b6b;
    --vvw-border-radius: 16px;
  }
</style>

<div id="gallery" class="my-gallery">
  <a href="/image1.jpg">
    <img src="/thumb1.jpg" alt="Photo 1" />
  </a>
</div>
```

## Responsive Styling

VistaView includes built-in responsive behavior, but you can customize it:

```css
/* Mobile overrides */
@media (max-width: 768px) {
  :root {
    --vvw-control-size: 40px;
    --vvw-border-radius: 4px;
  }
}

/* Desktop enhancements */
@media (min-width: 1024px) {
  :root {
    --vvw-control-size: 52px;
    --vvw-border-radius: 16px;
  }
}
```

## Extension Styles

Some extensions have their own stylesheets:

```js
// Image Story extension
import 'vistaview/style.css';
import 'vistaview/styles/extensions/image-story.css';
import { imageStory } from 'vistaview/extensions/image-story';

vistaView({
  elements: '#gallery a',
  extensions: [imageStory()],
});
```

## Dark Mode Support

Implement system-based dark mode:

```css
@media (prefers-color-scheme: dark) {
  :root {
    --vvw-bg-color: #000;
    --vvw-text-color: #fff;
    --vvw-accent-color: #4a9eff;
  }
}

@media (prefers-color-scheme: light) {
  :root {
    --vvw-bg-color: #fff;
    --vvw-text-color: #000;
    --vvw-accent-color: #007bff;
  }
}
```

## Accessibility Styles

VistaView respects user preferences:

```css
/* Reduced motion support (built-in) */
@media (prefers-reduced-motion: reduce) {
  .vvw-root * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}

/* High contrast support */
@media (prefers-contrast: high) {
  :root {
    --vvw-border-color: currentColor;
    --vvw-accent-color: #0066ff;
  }
}
```
